## <div align=center>GIF 轉換器<p>現行版本：1.0.0</p></div>
1. 將下載的壓縮包解壓縮到你想要的位置<br>
2. 將圖片放置於資料夾<br>  
> 支援的圖片格式有 <b>'.png', '.jpg', '.jpeg', '.bmp', '.tiff'</b>。<br>
> 支援多個圖片一起轉換。
<img src="./IMG_PREVIEW/step1.png"></img><br>
<img src="./IMG_PREVIEW/step1-1.png"></img><br>

3. 點擊 "點這個開始轉換.exe" 開始轉換圖片
> <img src="./IMG_PREVIEW/step2.png"></img><br>

4. 轉換後<br>
